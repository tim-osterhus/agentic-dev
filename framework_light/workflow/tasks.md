# Active Task
## Purpose
Single active card for this session. Only one task lives here at a time.

## How to use
1) Pull the top item from `workflow/tasks_backlog.md` into this file.  
2) Fill out the template below and keep it current as scope clarifies.  
3) When done, archive the card to `workflow/historylog.md` with outcomes and tests run.
